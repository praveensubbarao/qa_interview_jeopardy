# AI QE — Product UI Kit

A high-fidelity recreation of the AI QE quality engineering dashboard. Demonstrates sidebar nav, run list, run detail, agents panel, and analytics view as a click-through prototype.

## Components
- `Sidebar.jsx` — left nav with logo, sections, footer
- `Topbar.jsx` — context bar with search, environment switcher, profile
- `RunList.jsx` — table of runs with status badges, deltas, sparklines
- `RunDetail.jsx` — detail panel for a selected run
- `AgentCard.jsx` — live agent card with glow border
- `MetricCard.jsx` — analytics tile with metric + delta + sparkline
- `Button.jsx` — Primary/Secondary/Ghost/Gold variants

Open `index.html` for the interactive prototype.
